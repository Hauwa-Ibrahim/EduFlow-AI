# EduFlow AI Student Services Platform

## Overview

EduFlow AI is an enterprise Artificial Intelligence (AI) powered student services platform designed to automate the management of student loan applications. The platform combines AI, workflow automation, secure document management, and human review to improve efficiency, transparency, and decision-making.

## Features

- Student Registration and Login
- Student Loan Application
- Document Upload
- AI Document Verification
- AI Eligibility Assessment
- Human Review Workflow
- Email Notifications
- Workflow Automation (n8n)
- Secure PostgreSQL Database
- Reviewer and Administrator Dashboards
